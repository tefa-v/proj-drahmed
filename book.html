1 مقدمة المشروع

فكرة المشروع

هذا المشروع عبارة عن بوابة إلكترونية تعليمية تفاعلية لمادة اللغة العربية، تشرح درس "قصة أثر" بطريقة ممتعة وجذابة. يركز المشروع على عرض معلومات عن معلمين أثريين مهمين في مصر: الكنيسة المعلقة وقلعة قايتباي، مع اختبارات تفاعلية لقياس فهم الطالب.

التقنيات المستخدمة

التقنية الوظيفة
HTML5 بناء هيكل الصفحة وتنظيم المحتوى
CSS3 تصميم المظهر والألوان والتنسيق
JavaScript إضافة التفاعلية والحركة

أهداف المشروع

يهدف المشروع إلى تقديم تجربة تعليمية حديثة تجمع بين المعلومة والتفاعل، حيث يمكن للطالب قراءة المعلومات، مشاهدة الصور والفيديوهات، ثم اختبار نفسه للتأكد من فهمه للمحتوى.

ملاحظة مهمة

تم تصميم الكود بطريقة بسيطة ومناسبة للمبتدئين في البرمجة، مع استخدام تعليقات توضيحية تساعد على فهم كل جزء.

2 هيكل HTML الأساسي

ما هي HTML؟

HTML هي اختصار لـ HyperText Markup Language، وهي اللغة الأساسية لبناء صفحات الويب. تتكون من عناصر تسمى "وسوم" (Tags)، وكل وسم له وظيفة محددة في تنظيم المحتوى.

البنية الأساسية للصفحة

كل صفحة HTML تبدأ بالبنية التالية:

<!DOCTYPE html> <html lang="ar" dir="rtl"> <head> <meta charset="UTF-8"> <title>عنوان الصفحة</title> </head> <body> <!-- المحتوى هنا --> </body> </html>

الوسوم الأساسية المستخدمة

الوسم الوظيفة
header رأس الصفحة
section قسم مستقل
div حاوية عامة
h1, h2, h3 العناوين
p الفقرة النصية
img عرض الصور
video عرض الفيديو
button الأزرار

2 هيكل HTML (تكملة)

العناصر الهيكلية

عنصر header: يستخدم لرأس الصفحة، ويحتوي على العنوان الرئيسي والترحيب.

<header class="header"> <h1>البوابة الإلكترونية لمادة اللغة العربية</h1> <h3>شرح درس قصة أثر</h3> </header>

عنصر section: يستخدم لتجميع المحتوى المرتبط معاً في بطاقات مستقلة.

<section class="card"> <h2>أولاً: الكنيسة المعلقة</h2> <p>نص المحتوى...</p> </section>

صندوق الاختبار

يستخدم عنصر div مع input type="radio" لإنشاء أسئلة الاختيار من متعدد:

<div class="quiz-box"> <p>السؤال هنا</p> <input type="radio" name="q1"> الخيار الأول <input type="radio" name="q1"> الخيار الثاني </div>

معلومات مهمة

سمة class تُستخدم لتجميع العناصر المتشابهة، بينما سمة id تُستخدم لعنصر فريد وحيد في الصفحة.

3 تنسيقات CSS

ما هي CSS؟

CSS هي اختصار لـ Cascading Style Sheets، وهي اللغة المسؤولة عن مظهر صفحة الويب من ألوان وخطوط وتباعد وظلال وكل ما يتعلق بالتصميم البصري.

بنية قاعدة CSS

تتكون كل قاعدة CSS من ثلاثة أجزاء:

selector { property: value; }
الجزء الشرح مثال
المنتقي (Selector) يحدد العنصر .card
الخاصية (Property) ما نريد تغييره background-color
القيمة (Value) القيمة الجديدة blue

أنواع المنتقيات

/* منتقي العنصر */ body { } /* منتقي الفئة */ .card { } /* منتقي المعرف */ #header { } /* منتقي متعدد */ h1, h2, h3 { }

3 تنسيقات CSS (تكملة)

خصائص الألوان

.header { background-color: #2563eb; color: white; }

خصائص التباعد

.card { padding: 30px; /* المسافة الداخلية */ margin: 20px; /* المسافة الخارجية */ border-radius: 15px; /* الزوايا المستديرة */ }

خصائص النص

body { font-family: 'Tajawal', sans-serif; font-size: 16px; font-weight: bold; text-align: center; line-height: 1.8; }

الظلال

تُستخدم لإضافة عمق للعناصر:

.card { box-shadow: 0 10px 30px rgba(0, 0, 0, 0.1); } /* الظل: الإزاحة الأفقية، الإزاحة الرأسية، الضبابية، اللون */

نصيحة

استخدم rgba للألوان عندما تريد التحكم في الشفافية، حيث القيمة الأخيرة (من 0 إلى 1) تحدد درجة الشفافية.

3 تنسيقات CSS (تكملة)

التدرجات اللونية

تجعل التصميم أكثر احترافية وجاذبية:

.header { background: linear-gradient(135deg, #2563eb 0%, #1d4ed8 100%); } /* التدرج: الاتجاه، اللون الأول، النسبة، اللون الثاني، النسبة */

التصميم المتجاوب

يتكيف مع جميع أحجام الشاشات:

/* للشاشات الصغيرة (الهواتف) */ @media (max-width: 768px) { .header h1 { font-size: 1.8rem; } .card { padding: 20px; } }

الانتقالات

تجعل التغييرات سلسة:

.card { transition: all 0.3s ease; } .card:hover { transform: translateY(-5px); } /* عند المرور ترتفع البطاقة بسلاسة */

القيم الشائعة لـ transition

all: جميع الخصائص | 0.3s: المدة | ease: السلاسة

4 الألوان والمتغيرات

متغيرات CSS

المتغيرات تُخزن قيماً نستخدمها كثيراً، مما يسهل تغييرها في مكان واحد:

:root { --primary-color: #2563eb; --secondary-color: #dc2626; --accent-color: #059669; } .header { background-color: var(--primary-color); }

أخضر #059669 النجاح والأزرار
برتقالي #d97706 التنبيهات
رمادي #f1f5f9 الخلفية

6 التأثيرات والحركات

تعريف الحركة

@keyframes fadeInUp { from { opacity: 0; transform: translateY(30px); } to { opacity: 1; transform: translateY(0); } }

تطبيق الحركة

.card { animation: fadeInUp 0.6s ease forwards; } /* الحركة: الاسم، المدة، النوع، البقاء */

حركة الطفو

@keyframes float { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-10px); } } .header-icon { animation: float 3s ease-in-out infinite; }

أنواع الحركات الشائعة

ease: سلس | linear: ثابت | ease-in: بطيء ثم سريع | ease-out: سريع ثم بطيء

7 JavaScript والتفاعل

ما هي JavaScript؟

JavaScript هي لغة البرمجة للويب، تجعل الصفحة تفاعلية وحية. يمكنها تغيير المحتوى والرد على أحداث المستخدم مثل النقر والتمرير.

الدالة (Function)

function showChurchQuiz() { var quizBox = document.getElementById('church-quiz'); quizBox.style.display = 'block'; quizBox.scrollIntoView({ behavior: 'smooth' }); }

التحقق من الإجابة

function checkChurchQuiz() { var options = document.getElementsByName('church-q'); var resultBox = document.getElementById('church-result'); var answered = false; for (var i = 0; i < options.length; i++) { if (options[i].checked) { answered = true; if (options[i].value === "1") { resultBox.innerHTML = "أحسنت! إجابة صحيحة"; resultBox.className = "quiz-result correct"; } else { resultBox.innerHTML = "إجابة خاطئة"; resultBox.className = "quiz-result wrong"; } } } }

7 JavaScript (تكملة)

شريط التقدم

window.addEventListener('scroll', function() { var progressBar = document.getElementById('progressBar'); var scrollTop = window.scrollY; var docHeight = document.body.scrollHeight - window.innerHeight; var progress = (scrollTop / docHeight) * 100; progressBar.style.width = progress + '%'; });

زر العودة للأعلى

window.addEventListener('scroll', function() { var btn = document.getElementById('scrollTopBtn'); if (window.scrollY > 300) { btn.classList.add('visible'); } else { btn.classList.remove('visible'); } }); function scrollToTop() { window.scrollTo({ top: 0, behavior: 'smooth' }); }
